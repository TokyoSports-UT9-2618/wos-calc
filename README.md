# ホワイトアウトサバイバル 集結時間計算機

砦や工程ステーション、太陽城など拠点攻撃時に、対象までの距離（秒数）を入力するだけで正確な集結時間（集結のボタンを押す時間）を算出するためのツール。

## 主な機能
- 対象までの距離から集結開始時刻の自動計算
- カスタムメッセージの入力と出力（改行対応）
- メッセージの自動保存機能（LocalStorage）
- ワンクリックでの結果コピー

## 使い方
1. 残り時間を入力して「計算」ボタンを押すだけ
2. 集結主の名前・秒数・入力したメッセージ欄は一度入力するとブラウザに保存されます
3. 集結主の数を増やせます（テキストデータには3文字までで省略されます）
4. 秒数の欄を何も入力しない状態（文字を削除）すると計算結果にその集結主は出てきません。一時退席などで使えます。
5. 集結主・秒数を入力後、1分・5分・10分の集結時間をボタンで選択してください
6. 選択した時間居合わせて現在時刻から自動で計算し、クリップボードにコピーしてくれます
7. ゲームに戻ってペーストするだけで、計算結果がチャットで送れます
8. 着弾時間を統一するための計算も可能です
9. その際は希望する着弾時間をHH:MM:SSで入力して「計算＆コピー」ボタンを押すだけ
10. カスタムメッセージには「着弾時間に合わせて派兵してください」や「XXXにはFC10のみ乗せてください」などの指定が出来ます。
